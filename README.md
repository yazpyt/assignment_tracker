# Assignment Tracker

A simple desktop application to help students track assignments with local JSON storage.

## Features
- Add, edit, and track assignments
- Progress bars for completion tracking
- Sort by deadlines
- Calendar date pickers
- Local data storage

## Usage
1. Click "Add Subject" to create new entries

2. Edit any field directly in the table

3. Click "Save" to persist changes

4. Data is stored in assignments.json
